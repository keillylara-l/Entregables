## 📋 Descripción

Este repositorio es el portafolio digital del estudiante para el curso de Econometría. Sirve para documentar el trabajo realizado, evidenciar las habilidades adquiridas a lo largo del semestre y demostrar el uso efectivo de herramientas como Git y GitHub en un contexto académico cuantitativo.



## 📦 Contenido

| Elemento | Descripción |
|---|---|
| **Scripts R** | Archivos `.R` y scripts de análisis econométrico |
| **R Notebooks / RMarkdown** | Talleres y Trabajos del curso |
| **Datasets** | Conjuntos de datos utilizados en análisis y proyectos |
| **Informes** | Documentos y reportes de entregas formales |
